# WeatherCast
WeatherCast is a modern, interactive weather app built with React Native. It provides real-time weather forecasts using the Open Meteo API, featuring smooth animations, responsive design, and a clean user interface. Users can easily check current weather conditions, hourly forecasts, and upcoming trends with a seamless experience.
